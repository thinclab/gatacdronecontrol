Detailed Description

Author:
Vince Capparell, Casey Hetzler
Version:
1.0

A program designed to allow one server to control 1-3 quadcopter clients

in a grid-based environment, providing various functions such as movement, communication of navdata, and the spotting of other drones.

DESCRIPTION

GaTACDroneControl allows grid-based movement, facilitated with client-server communication, for up to three (3) "Parrot AR.Drone" drone clients and one (1) multi-threaded server process. Made for cooperative use with UGA THINC Lab's "ardrone_thinc" package and Autonomy Lab's "ardrone_autonomy" package.
